US Accidents Dashboard Documentation

The purpose of the visualization is to get an understanding of the distribution of accidents and their severity across the US. To achieve this purpose
Plotly Dash was used to create a basic dashboard layout and graph elements. Linking these is made easier by using Plotly Dash as well. Polars is used
to handle the large amount of data and multiple parquet files split the data into digestable chunks for the dashboard.

Usage works via interaction with the charts as well as the dropdown menus that can be used for filtering and looking at different indicators.

To run the application simply download all files and execute the requirements.txt using the command pip install -r requirements.txt.
If this runs succesfully and you have downloaded all data files you can run the app by excuting the app.py file with the command python app.py.
Should not all data be present simply execute the preprocessing.py after having downloaded the full dataset from https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents
and placing the file in the data folder. This will create all relevant files.

All source code and documentation can be found under: https://github.com/JonasUnruh/RealtimeVis
